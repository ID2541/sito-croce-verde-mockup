## Pattern adottati nel frontend
- Navigazione top-level chiara con sezioni operative e accesso rapido a prenotazione.
- Hero + Feature Grid + News List per homepage istituzionale.
- Page Header con breadcrumb per orientamento IA.
- Card list e dettagli dinamici per servizi/news (`[slug]`).
- Form layout compatti e accessibili per contatti e prenotazione (UI ready, submit disabilitato).
